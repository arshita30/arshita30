## What I work on

**EmotiCode**
Fine-tuned Gemma-3 with QLoRA for emotion classification in low-resource Indic languages, Santali, Kashmiri, and Manipuri. Most NLP tooling skips these languages entirely, so a lot of the work was data prep and evaluation, not just training.

**Legacy Codebase MCP Server**
Built an MCP server to help debug legacy TPF and Java codebases, integrated with VS Code and GitHub Copilot. Point is to give LLMs useful context on old, undocumented systems instead of guessing.

**ALICE: Vector DB Migration**
Migrated a production vector database from ChromaDB to Alloy. Mostly about not breaking retrieval quality while moving infra underneath it.

**LLM Document Generation Pipeline**
System for generating HLDs and BRDs using LLMs, built to cut down the manual time spent on structuring technical docs.
